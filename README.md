# Drone Simulator

## Tech Stack

**Client:** React

**API:** Maps JavaScript API (by Google Developer Console)


## Run Locally

Clone the project

```bash
  git clone git@github.com:utkrao/drone-simulator.git
```

Go to the project directory

```bash
  cd drone-simulator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
